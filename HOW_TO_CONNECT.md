# How to Connect 2 Players in Same Session

## Quick Guide

### Method 1: Using Room Codes (Recommended)

1. **Player 1 (Host):**
   - Open Tic Tac Toe game
   - Click "Create Online Room"
   - Enter your name (for chat)
   - Select grid size (3×3, 4×4, or 5×5)
   - Room code will be generated (e.g., "ABC123")
   - Share this code with Player 2

2. **Player 2 (Guest):**
   - Open Tic Tac Toe game
   - Click "Join Online Room"
   - Enter the room code shared by Player 1
   - Click "Join"
   - Game will start automatically!

### Method 2: Using Global Join Button

1. **Player 1:** Creates room and gets code
2. **Player 2:** 
   - Click the gamepad icon button (bottom-right)
   - Enter the room code
   - Game automatically switches and joins!

### Method 3: Using Public Rooms Browser

1. Click the Wi-Fi icon in header
2. Browse available public rooms
3. Click "Join" on any available room
4. Game starts instantly!

## Chat & Reactions

Once both players are connected:
- **Chat:** Click the chat icon in the slideable panel (right side)
- **Reactions:** Click the smiley icon to send emoji reactions
- Both features work in real-time during the game!

## Features Available

✅ Real-time game synchronization
✅ Live chat messaging
✅ Emoji reactions
✅ Turn timer (15 seconds)
✅ Score tracking
✅ Auto-reconnect on disconnect

## Troubleshooting

- **Can't join?** Make sure the room code is correct (6 characters, case-insensitive)
- **Room not found?** The room might have ended or the code is wrong
- **Connection issues?** Check your internet connection and Supabase configuration
